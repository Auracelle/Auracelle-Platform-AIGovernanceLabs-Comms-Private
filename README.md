# Auracelle · Institutional Briefing Pack (CONFIDENTIAL)

This repository hosts a single-page HTML briefing pack for **Auracelle AI Governance Labs**.

## What’s inside
- `index.html` — the full interactive/scrollable briefing pack (self-contained, includes embedded logo as base64).

## Publish with GitHub Pages
1. Push this repository to GitHub.
2. In GitHub: **Settings → Pages**
3. Under **Build and deployment**, select **GitHub Actions**.
4. The included workflow will deploy `index.html` to Pages on every push to `main`.

## Confidentiality
This material is **not for public distribution** unless you explicitly intend to publish it.
If you need a public-safe version, create a separate redacted repository.
